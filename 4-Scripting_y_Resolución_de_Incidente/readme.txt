Scripting y Resolución de Incidente

Objetivo: Evaluar capacidad de automatizar tareas y actuar ante incidentes.

Requisitos:
•	Crear un script (Bash, PowerShell o Python) para automatizar una tarea típica (reinicio, limpieza, verificación de conectividad, etc.).
•	Resolver un caso simulado a partir de logs o archivos con errores. Documentar causa raíz y solución.
